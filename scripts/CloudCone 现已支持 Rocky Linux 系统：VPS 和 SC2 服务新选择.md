# CloudCone 现已支持 Rocky Linux 系统：VPS 和 SC2 服务新选择

我们很高兴地宣布，**Rocky Linux 现已正式登陆 CloudCone 云服务平台**。作为 CentOS 的替代发行版，Rocky Linux 以其稳定性和企业级特性受到开发者青睐。现在，您可以在 CloudCone 的 VPS 和 SC2 服务器上轻松部署这款优秀的 Linux 系统。

## 如何在 CloudCone 上使用 Rocky Linux？

使用 Rocky Linux 模板非常简单：

1. 创建新服务器时，在操作系统选项中选择 **Rocky Linux**
2. 根据需求选择服务器配置（CPU、内存、存储等）
3. 选择数据中心位置
4. 完成其他常规设置
5. 启动服务器即可

服务器运行后，您可以像使用其他 Linux 发行版一样操作 Rocky Linux，享受其出色的稳定性和兼容性。
